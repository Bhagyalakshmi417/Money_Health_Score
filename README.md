# 💰 Money Health Score

## 🚀 Overview

An AI-powered financial health assessment tool that evaluates users across 6 key financial dimensions and provides actionable insights.

## 🎯 Problem

Most individuals lack a structured financial plan and awareness of their financial health.

## 💡 Solution

This app calculates a personalized Money Health Score and gives recommendations based on user inputs.

## 🛠️ Tech Stack

* React (Vite)
* JavaScript
* Recharts
* CSS

## ✨ Features

* Multi-step financial form
* Score calculation engine
* Visual dashboard (radar chart)
* AI-based financial advice (fallback included)

## ▶️ How to Run

```bash
npm install
npm install recharts
npm run dev
```
